### API Rate Limiting

**Problem:** Leaguepedia API hat strenge Rate Limits
- Empfohlene Verz√∂gerung: **30+ Sekunden** zwischen Anfragen
- Mit Bot-Authentifizierung: **3 Sekunden** (siehe `.env`-Konfiguration)
- Bei √úberschreitung: Exponentielles Backoff (3s, 6s, 12s)

**L√∂sung:** Verwende das bestehende `LeaguepediaLoader`-System:
```python

```

### URL-Format

**WICHTIG:** Leaguepedia verwendet **Leerzeichen**, nicht Underscores!

**Richtig:**
```
LEC/2024 Season/Spring Season
2024 Season World Championship/Play-In
```

**Falsch:**
```
LEC/2024_Season/Spring_Season
2024_Season_World_Championship/Play-In
```

### Season 3 Spezialfall

2013 verwendet teilweise "Season 3" statt "2013 Season":
